## 掌握窗口的管理









## vim
### 新建窗口
  - 左右 -> Ctrl-w v
  - 上下 -> Ctrl-w s

### 窗口切换
  - Ctrl-w hjkl
  - Ctrl-w w

### 关闭窗口
  - Ctrl-w c

## vscode
  - 新建窗口 
    - Command + \
    - Command + Ctrl + \
  
  - 关闭窗口
    - command + w
    - command + k + w
  
  - 窗口切换
    - shift + 方向键 ->需要改键
    - shift + ctrl + jkhl
