

## 发现使用快捷键的场景

## 显示/隐藏资源资源管理器
    - command + b

## 使用live Server启动服务
    - command + l command + o -> 启动服务
    - command + l command + c -> 关闭服务

## 预览md
    - command + k + v
    - command + shift + v

## show extensions 
    - command + shift + x

## 打开当前所在文件夹
    - command + k + r

## 复制当前路径
    - command + k + p
