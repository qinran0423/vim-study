## 宏 ->  可以录制一系列动作
## 录制
  - q a -> a 代表寄存器的名字 q 开始录制
  - q -> q 结束录制
## 查看 -> :reg【空格】a
## 使用 
  - @a -> a是寄存器的名字
  - @@ -> 调用最后一个执行的宏
  - 重复执行 -> 数字+ @ + 寄存器 -> 10@a

安全机制 -> 报错就停
